# 🚀 AI Deadline Hero

> Never Miss a Deadline Again.

AI Deadline Hero is an AI-powered productivity assistant that helps students and professionals organize tasks, plan work intelligently, and stay productive before deadlines.

---

## ✨ Features

✅ Smart Task Management

- Add tasks
- Delete tasks
- Mark tasks as completed
- Automatic task saving using JSON

✅ AI Productivity Planner

- AI-generated task breakdown
- Priority suggestion
- Daily schedule
- Productivity tips
- Motivation

✅ Dashboard

- Total Tasks
- Completed Tasks
- Pending Tasks
- Productivity Score

✅ Analytics

- Pie Chart
- Bar Chart
- Progress Tracking

✅ Smart Features

- Deadline countdown
- Overdue detection
- Search tasks
- Modern responsive UI
- Dark Theme

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Deadline-Hero.git
```

Open project

```bash
cd AI-Deadline-Hero
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the app

```bash
streamlit run app.py
```
